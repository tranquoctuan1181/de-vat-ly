Cài đặt Google Sheets
Dành cho giáo viên · Chỉ làm 1 lần duy nhất
Các bước thực hiện
1
Mở sheets.new → đặt tên file "Kết quả kiểm tra"
2
Vào menu Extensions → Apps Script → xoá code cũ → dán toàn bộ nội dung file google_apps_script.js → Save
3
Nhấn Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone → Deploy
4
Copy URL dạng https://script.google.com/macros/s/…/exec → dán vào ô bên dưới
Chia sẻ file HTML cho học sinh qua Zalo:
Zalo → Nhóm lớp → nhấn biểu tượng File (📎) → chọn file HTML → Gửi.
HS nhận file → nhấn giữ → chọn "Mở bằng trình duyệt" (Chrome/Safari) → làm bài bình thường.
